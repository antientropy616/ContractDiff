# 合同对比助手 v1.2.3
## Contract Diff Tool

一款专业的合同文档差异对比工具，专为法律工作场景设计。

---

## 📋 功能特点

- ✅ **自动对比** - 自动识别两个 Word/PDF 文档之间的差异
- ✅ **详细改动列表** - 清晰展示所有文字增删改
- ✅ **对比报告** - 生成独立的 Excel 改动清单
- ✅ **PDF 支持** - 支持 PDF 文档对比（需安装额外依赖）
- ✅ **表格识别** - 识别表格内容的新增、删除、修改
- ✅ **图片识别** - 识别图片的新增和删除
- ✅ **页眉页脚** - 识别页眉页脚的改动
- ✅ **脚注识别** - 识别脚注内容的改动
- ✅ **日志系统** - 自动记录运行日志，方便问题排查
- ✅ **异常处理** - 友好的错误提示，完善的异常捕获
- ✅ **自动更新** - 定期检查新版本，一键跳转下载
- ✅ **本地处理** - 100% 离线运行，文档不上传云端
- ✅ **专业界面** - 法律严肃风格，符合法务工作习惯

---

## 🚀 快速开始

### 方法一：直接使用 exe（推荐）

1. 双击运行 `合同对比助手.exe`
2. 选择原文档 (X) 和修改文档 (Y)
3. 选择输出选项
4. 点击"开始对比"
5. 查看结果文档

### 方法二：从源码运行

```bash
# 1. 安装 Python 3.10+
# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行程序
python src/main.py
```

## 📦 构建说明

### Windows 环境

```bash
# 安装依赖
pip install -r requirements.txt

# 执行构建
python build.py

# 生成的 exe 位于：
# - dist/合同对比助手.exe
# - ~/Documents/合同对比助手.exe
```

### macOS/Linux 环境建议直接去macOS的branch

⚠️ **注意**: 当系统为 macOS，无法直接生成 Windows exe 文件。

有以下解决方案：

1. **在 Windows 电脑上构建**
   - 将整个项目文件夹复制到 Windows 电脑
   - 运行 `python build.py`

2. **使用交叉编译工具**
   - 需要配置 Wine 环境
   - 复杂度较高，不推荐

3. **使用虚拟机**
   - 在 macOS 上运行 Windows 虚拟机
   - 在虚拟机中执行构建

## 📝 使用说明

### 1. 选择文档

- 点击"选择文件..."按钮
- 支持 Word (.docx) 和 PDF (.pdf) 格式
- 或拖拽文件到选择区域
- 支持 Word 2007+ 格式 (.docx)

### 2. 选择输出选项

| 选项 | 说明 |
|------|------|
| 详细改动列表 | 清晰展示所有文字增删改 |
| 对比报告 | 生成独立的 Excel 清单 |

### 3. 开始对比

- 点击"开始对比"按钮
- 等待处理完成（通常<30 秒）
- 查看结果文档

---

## ⚠️ 注意事项

1. **文档格式**: 支持 .docx 和 .pdf 格式，不支持 .doc
2. **文件大小**: 建议单文件 ≤ 50MB
3. **PDF 处理**: PDF 对比需要安装额外依赖（pdfplumber、pdf2docx）
4. **数据安全**: 所有处理在本地完成，不联网（自动更新检查除外）
5. **结果复核**: 建议人工复核重要改动
6. **日志文件**: 运行日志位于 `~/Documents/合同对比助手/logs/`
