## 📦 如何构建 macOS 版

```bash

cd /Users/yuclaw/.openclaw/workspace/合同对比助手

# 使用虚拟环境构建
./venv/bin/python3 build_mac.py

# 或使用系统 Python（如果已安装依赖）
python3 build_mac.py
```

## 📝 注意事项

### 首次运行
macOS 会提示"无法打开，因为来自身份不明的开发者"

**解决方法：**
1. 系统偏好设置 → 安全性与隐私
2. 点击"仍要打开"
3. 之后可正常使用

### 依赖安装
构建前确保安装依赖：
```bash
pip3 install -r requirements.txt
```

或使用项目虚拟环境：
```bash
./venv/bin/pip3 install -r requirements.txt
```
