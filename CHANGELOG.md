# 合同对比助手 - 版本更新日志

## v1.2.3 (2026-03-04)

### 🐛 关键 Bug 修复

1. **启动崩溃修复**
   - 修复：`doc_comparator.py` 类定义缩进错误导致 `IndentationError`
   - 影响：程序无法启动，打包后必现崩溃

2. **页眉页脚提取修复**
   - 修复：`'_Header' object has no attribute 'text'` 错误
   - 方案：改用 `section.header.paragraphs` 遍历提取

### ✨ 体验优化

1. **位置显示优化**
   - 优化：详细改动列表中位置从「位置：0」改为「第 1 段」
   - 影响：更直观，用户无需计算段落序号

2. **署名信息**
   - 新增：关于对话框增加「玉哥与他的虾」署名
   - 意义：纪念共同开发历程

3. **版本管理**
   - 优化：版本号从 `version.json` 动态读取
   - 好处：更新版本无需修改代码

### 🧹 项目清理

删除以下开发阶段临时文件：
- `verify_code.py` - 代码验证脚本
- `test_image_recognition.py` - 图片功能测试
- `test_table_recognition.py` - 表格功能测试
- `run_tests.py` - 简单测试运行器
- `build_debug.py` - 调试构建脚本（功能已合并至 build.py）
- `contract_diff_debug.spec` - 调试版 spec（可动态生成）
- `闪退问题诊断指南.md` - 临时调试文档
- `v1.1_迭代报告.md` - 历史迭代报告

## v1.2 (2026-03-03)

### 🛠️ 新增功能

1. **日志系统**
   - 自动记录运行日志到 `~/Documents/合同对比助手/logs/`
   - 日志按天轮转，单文件最大 5MB，保留 3 个历史文件
   - 支持调试、信息、警告、错误等级别
   - 方便问题排查和追踪

2. **异常捕获**
   - 完善的异常分类体系（文件加载、格式、对比、输出、网络等）
   - 友好的错误提示信息
   - 详细的错误日志记录
   - 提升用户体验和稳定性

3. **单元测试**
   - 核心功能测试覆盖（对比器、版本比较、日志、错误处理、PDF 处理器）
   - 自动化测试框架
   - 运行测试：`python tests/test_core.py`
   - 保证代码质量和回归测试

4. **自动更新**
   - 启动时自动检查新版本（每 7 天一次）
   - 手动检查更新（工具菜单）
   - 更新提示和下载链接
   - 方便版本升级

5. **PDF 支持**
   - 支持 PDF 文档对比（需安装 pdfplumber 和 pdf2docx）
   - PDF 自动转换为 Word 格式进行对比
   - 文件选择支持 PDF 格式筛选
   - 扩展使用场景

6. **更新说明**
   - 帮助菜单新增【更新说明】项
   - 查看完整历史版本变更日志
   - 了解每次版本更新的具体内容

### 📦 依赖更新

- 新增：`pdfplumber>=0.9.0`（PDF 文本提取）
- 新增：`pdf2docx>=0.5.6`（PDF 转 Word）
- 新增：`requests>=2.28.0`（网络请求）
- 新增：`pytest>=7.0.0`（测试框架）

### 🐛 Bug 修复

- 修复临时文件未清理的问题
- 优化错误处理流程
